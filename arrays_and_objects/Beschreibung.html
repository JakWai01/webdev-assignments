C7: Optional: Ein kleines Spiel

Die Vorlage enthlt ein kleines halb-fertiges Spiel.

Es fliegen graue Kreise von rechts nach links. Wenn man klickt soll eine Bombe an der geklickten Position in die Welt gesetzt werden die nach 1 Sekunde explodiert. Dabei soll sie alle grauen Kreise in der Umgebung zerstren. Ziel ist es dass so wenig graue Kreise wie mglich die linke Seite erreichen.

Ergnzen Sie die noch fehlende grundlegende Funktionalitt:

  • In der onGameClick()-Funktion soll eine neue Bombe in die Welt gesetzt werden. Eine Bombe ist dabei das folgende Objekt:

    {
    type: 'bomb', onDraw: onBombDraw,
    secondsUntilExplosion: 1.0, onExplosion: onBombExplosion,
    x: x, y: y, radius: 50
}

    Dieses neue Objekt muss zu dem Array gameObjects (eine globale Variable) hinzugefgt werden. Dieses Array enthlt alle Objekte in der Spielewelt.

  • Sobald eine Bombe explodiert wird vom Spiel die Funktion onBombExplosion() aufgerufen. Vervollstndigen Sie diese Funktion damit sie:

    • Alle grauen Kreise findet (Objekte in gameObjects-Array mit der type-Property von target) die im Kreis der Bombe sind.

      Verwenden sie die Funktion gameObjectsIntersect(bomb, target) um zu testen ob ein target im Kreis der Bombe ist.

    • Die gefundenen grauen Kreise zu dem Array gameObjectsThatDiedThisFrame hinzufgt (eine globale Variable). Damit verschwinden die grauen Kreise am Ende des Frames aus der Spielewelt.

      Mit der Funktion arrayA = arrayA.concat(arrayB) knnen sie Elemente eines Arrays an ein anderes anhngen.

  • Have Fun! Bauen Sie das Spiel um wie es Ihnen gefllt. Sie knnen auch den Code in game.js nach belieben verndern. Hier ein paar Ideen:

    • Ein Zhler einbauen der zhlt wie viele grauen Kreise man mit Bomben erwischt hat.
    • Madness-Mode: spawnTargetEveryNSeconds in game.js auf 1 / 30 ndern (erzeugt einen grauen Kreis alle 1/30 Sekunden). Oder gar 1/100.
    • Dem Spieler nur erlauben 2 Bomben pro Sekunde zu plazieren.

    • Das Spiel beenden sobald eine gewisse Anzahl an grauen Kreisen die linke Seite erreicht haben.

      Die Funktion onTargetLeave() wird aufgerufen sobald ein grauer Kreis die link Seite erreicht.

    • Neue Game-Objects in die Welt setzen die sich in irgendwelche Richtungen bewegen.

      Damit sich ein Game-Object bewegt muss es zumindest die Properties x, y, velocityX und velocityY haben.

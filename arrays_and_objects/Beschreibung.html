C Arrays and Objects

C1: Einen Artikel eines kleinen Online-Shops anzeigen

Um auf die Artikel-Daten des Online-Shops zuzugreifen müssen sie das Skript articles.js einbinden. Danach stehen ihnen die Artikel in der Variable articles zur Verfügung.

  • Geben Sie zuerst den 4. Artikel (der Artikel mit dem Index 3) per console.log() in der Browser-Konsole aus.
  • Geben Sie nur den Namen diese Artikels in der Browser-Konsole aus.

  • Generieren Sie HTML-Code der alle Properties und deren Werte des Artikels anzeigt. Dabei sollen alle Properties des Objekts angezeigt werden, auch welche die evt. erst in Zukunft hinzugefügt werden.

    Der HTML-Code sollte dabei folgender maßen aussehen (für jede Property soll ein Paar an dt- und dd-Elementen erscheinen):

    <dl>
    <dt>type</dt><dd>…</dd>
    <dt>name</dt><dd>…</dd>
    …
</dl>

    Verwenden sie document.querySelector('main').innerHTML += 'new-html-code' um neuen HTML-Code zu dem <main>-Element der Vorlage hinzuzufügen.

C2: Alle Artikel des Shops anzeigen

Zeigen Sie alle Artikel des Online-Shops an. Jeder einzelne Artikel soll dabei wie in Aufgabe 1 ausgegeben werden.

Zur besseren Abgrenzung können Sie alle Artikel in einer ul-Liste anzeigen oder nach jedem Artikel ein hr-Element ausgeben.

C3: Artikel sortieren

  • Sortieren Sie alle Artikel nach Preis (billigste Artikel zuerst) und geben Sie sie wie in Aufgabe 2 aus.

    Sie können in einer Sortier-Funktion den Minus-Operator verwenden um nach Zahlenwerten zu sortieren.

  • Geben Sie alle Artikel sortiert nach Namen aus.

    Sie können die Funktion oneString.localeCompare(otherString) verwenden um nach Strings zu sortieren.

C4: Artikel filtern und Preis aufsummieren

  • Erstellen Sie ein neues Array das alle Notebooks enthält (Artikel mit type notebook).

    Um das neue Array zu prüfen können Sie es mit console.log() ausgeben.

  • Berechnen Sie die Summe aller Preise dieser Artikel (was es kosten würde wenn man jedes Notebook ein mal kauft). Geben Sie die Summe mit console.log() aus.

  • Erstellen sie ein neues Array dass nur die Namen dieser Notebooks enthält. Geben Sie das Array mit console.log() aus.

    Sie können die Array.map()-Funktion verwenden um das neue Array zu erstellen.

C6: Artikel hinzufügen und löschen

Ein Artikel muss kurzfristig aus dem Angebot genommen werden. Dafür muss ein neuer Artikel hinzugefügt werden. Leider ist die einzige Person, die mit dem Datenbank-Stack vertraut ist, gerade erkrankt. Daher müssen Sie die Artikel in ihrem Code direkt einpflegen.

  • Entferen Sie den Artikel mit dem Index 2 aus dem articles-Array (ohne articles.js zu bearbeiten).

    Aufgrund eines Patent-Streits ist es der Firma "Apple Computer, Inc." kurzfristig untersagt Tablets mit einem speziellen Eck-Radius zu verkaufen.

  • Fügen Sie folgenden Artikel zum articles-Array hinzu (am Ende):

    price: 400,
    display: '9.7", 2048x1536',
    os: 'iPadOS 13.2.3',

}

C7: Optional: Ein kleines Spiel

Die Vorlage enthält ein kleines halb-fertiges Spiel.

Es fliegen graue Kreise von rechts nach links. Wenn man klickt soll eine Bombe an der geklickten Position in die Welt gesetzt werden die nach 1 Sekunde explodiert. Dabei soll sie alle grauen Kreise in der Umgebung zerstören. Ziel ist es dass so wenig graue Kreise wie möglich die linke Seite erreichen.

Ergänzen Sie die noch fehlende grundlegende Funktionalität:

  • In der onGameClick()-Funktion soll eine neue Bombe in die Welt gesetzt werden. Eine Bombe ist dabei das folgende Objekt:

    {
    type: 'bomb', onDraw: onBombDraw,
    secondsUntilExplosion: 1.0, onExplosion: onBombExplosion,
    x: x, y: y, radius: 50
}

    Dieses neue Objekt muss zu dem Array gameObjects (eine globale Variable) hinzugefügt werden. Dieses Array enthält alle Objekte in der Spielewelt.

  • Sobald eine Bombe explodiert wird vom Spiel die Funktion onBombExplosion() aufgerufen. Vervollständigen Sie diese Funktion damit sie:

    • Alle grauen Kreise findet (Objekte in gameObjects-Array mit der type-Property von target) die im Kreis der Bombe sind.

      Verwenden sie die Funktion gameObjectsIntersect(bomb, target) um zu testen ob ein target im Kreis der Bombe ist.

    • Die gefundenen grauen Kreise zu dem Array gameObjectsThatDiedThisFrame hinzufügt (eine globale Variable). Damit verschwinden die grauen Kreise am Ende des Frames aus der Spielewelt.

      Mit der Funktion arrayA = arrayA.concat(arrayB) können sie Elemente eines Arrays an ein anderes anhängen.

  • Have Fun! Bauen Sie das Spiel um wie es Ihnen gefällt. Sie können auch den Code in game.js nach belieben verändern. Hier ein paar Ideen:

    • Ein Zähler einbauen der zählt wie viele grauen Kreise man mit Bomben erwischt hat.
    • Madness-Mode: spawnTargetEveryNSeconds in game.js auf 1 / 30 ändern (erzeugt einen grauen Kreis alle 1/30 Sekunden). Oder gar 1/100.
    • Dem Spieler nur erlauben 2 Bomben pro Sekunde zu plazieren.

    • Das Spiel beenden sobald eine gewisse Anzahl an grauen Kreisen die linke Seite erreicht haben.

      Die Funktion onTargetLeave() wird aufgerufen sobald ein grauer Kreis die link Seite erreicht.

    • Neue Game-Objects in die Welt setzen die sich in irgendwelche Richtungen bewegen.

      Damit sich ein Game-Object bewegt muss es zumindest die Properties x, y, velocityX und velocityY haben.
